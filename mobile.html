Project Revenge

 So what is "Project Revenge"?
It is a complete rewrite of the popular "TotalFreedomMod" developed by Madgeek1450 (Steven Lawson) and Prozza (DarthSalamon) for the TotalFreedom Minecraft Server. You can read more about it here.

Project Revenge's goal is to fulfill the same basic capabilities that TotalFreedomMod has but also to expand on the features to improve stability and fuctionality.

See the other categories for functionality of the sub-projects.

Project Alpha

Project Alpha is the main focus. It is the Minecraft Plugin that will hold everything together.

Project Beta

Project Beta adds Telnet capabilities to Project Alpha.

Project Gamma

 Project Gamma is the Official Discord Bot for Project Revenge.

Project Delta

 Project Delta is a Gamma expansion. This will allow Discord and Minecraft to communicate, letting the user talk both ways. This is both a bot and a plugin.
